
千星奇域的月之八版静态、动态像素画


【用于】

1、png图片序列生成gia实体场景动画组

2、png转静态像素画

**纯粹的小实验产物，无实用性

*动画仅支持最大30x30，30帧存储，此时关卡服务器存档已经耗尽

*静态像素画最大分辨率100 * 100内建议，并且只是生成一对一像素的静态实体，并没有合并重复像素的

【前置】

环境依赖：安装Python

程序所用版本 3.14.5，其它版本也行不建议太低

命令提示符命令行运行py

提示Python 3.14.5 版本号成功

安装图像库：pip install Pillow

GIA解封装，在主页另一个gia-protobuf项目内

使用请看相关文档

===========建议所有测试都在空存档里进行==========

无误后从空白存档导出再导入目标存档
